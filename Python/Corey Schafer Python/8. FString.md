### Python Quick Tip: F-Strings - How to Use Them and Advanced String Formatting

#### Key Insights

- **Introduction to F-Strings**
  - F-strings, introduced in Python 3.6, are a more intuitive and readable way to format strings.
  - Preferred over older formatting methods like `%` operator and `str.format()`.

- **Basic Usage**
  - Syntax: Prefix the string with `f` and include variables within curly braces `{}`.
  - Example:
    ```python
    first_name = "John"
    last_name = "Doe"
    print(f"My name is {first_name} {last_name}")
    ```
  
- **Comparison with `str.format()` Method**
  - Old method:
    ```python
    print("My name is {} {}".format(first_name, last_name))
    ```
  - F-strings are more readable and eliminate the need to match placeholders with variables manually.

- **Embedding Expressions**
  - Functions or methods can be called within the curly braces.
  - Example:
    ```python
    print(f"My name is {first_name.upper()} {last_name.upper()}")
    ```

- **Accessing Dictionary Values**
  - Directly access dictionary values within F-strings.
  - Example:
    ```python
    person = {"name": "Jen", "age": 23}
    print(f"My name is {person['name']} and I am {person['age']} years old")
    ```
  - Note: Use double quotes for the string if single quotes are needed inside curly braces.

- **Performing Calculations**
  - Perform calculations directly within F-strings.
  - Example:
    ```python
    print(f"Four times eleven is {4 * 11}")
    ```

- **Zero Padding**
  - Pad numbers with leading zeros for formatting purposes.
  - Example:
    ```python
    for n in range(1, 11):
        print(f"The value is {n:02}")
    ```

- **Formatting Floating Point Numbers**
  - Control the precision of floating-point numbers.
  - Example:
    ```python
    pi = 3.141592653589793
    print(f"Pi to four decimal places: {pi:.4f}")
    ```

- **Date Formatting**
  - Format dates using F-strings with `datetime` formatting codes.
  - Example:
    ```python
    from datetime import datetime
    birthday = datetime(1990, 1, 1)
    print(f"Jen has a birthday on {birthday:%B %d, %Y}")
    ```
